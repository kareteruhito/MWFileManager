# MWFileManager

MWFileManager は、WPF / C# で作成したシンプルなファイルマネージャです。  
ファイルマネージャの基本機能を実装した学習・拡張用のベースプロジェクトです。

## 概要

左側にドライブ・フォルダのツリー、右側に現在のディレクトリ内のファイル一覧を表示します。  
フォルダ移動、ファイルの関連付け起動、コピー、移動、削除、貼り付け、名前順ソートなどの基本操作に対応しています。

## 主な機能

- ドライブ一覧の表示
- フォルダツリーの表示
- アドレスバーによるディレクトリ移動
- 親ディレクトリへの移動
- ファイル・フォルダ一覧の表示
- Windows のシステムアイコン表示
- フォルダのダブルクリック移動
- ファイルのダブルクリック起動
- 右クリックメニュー
  - コピー
  - 移動
  - 削除
  - 貼り付け
- 同名ファイルが存在する場合の連番コピー・移動
  - 例: `sample.txt` → `sample(1).txt`
- 複数選択
- ディレクトリ / ファイルの並べ替え
- 名前列ヘッダーによるソート

## 開発環境

- C#
- WPF
- .NET 10.0 Windows
- Visual Studio Code

## 必要環境

Windows 環境での実行を想定しています。  
プロジェクトは WPF を使用しているため、Windows 以外の OS では動作しません。

## ビルド方法

リポジトリをクローンします。

```bash
git clone https://github.com/ユーザー名/MWFileManager.git
```

プロジェクトフォルダへ移動します。

```bash
cd MWFileManager
```

Visual Studio で `MWFileManager.csproj` を開き、ビルドして実行します。

または、.NET SDK が利用できる環境で次のコマンドを実行します。

```bash
dotnet build
dotnet run
```

## 基本操作

### フォルダを開く

左側のツリーからフォルダを選択するか、右側の一覧でフォルダをダブルクリックします。

### ファイルを開く

右側の一覧でファイルをダブルクリックすると、Windows の関連付けアプリで開きます。

### パスを指定して移動する

アドレスバーにディレクトリパスを入力し、移動ボタンをクリックします。

### コピー・移動・削除

右側のファイル一覧で対象を選択し、右クリックメニューから操作します。  
複数選択にも対応しています。

### 貼り付け

コピーまたは移動を実行したあと、移動先のディレクトリで右クリックメニューから「貼り付け」を選択します。

## 実装メモ

### MainWindow.xaml

画面レイアウトを定義しています。  
`TreeView`、`ListView`、`Menu`、`StatusBar`、アドレスバーなどを配置しています。

### MainWindow.xaml.cs

ファイルマネージャの主要なイベント処理を実装しています。

主な処理:

- フォルダツリー初期化
- ディレクトリ移動
- ListView のダブルクリック処理
- コピー / 移動 / 削除 / 貼り付け
- CollectionView によるソート

### IconHelper.cs

Windows の Shell API を使用して、ファイルやフォルダのシステムアイコンを取得します。
